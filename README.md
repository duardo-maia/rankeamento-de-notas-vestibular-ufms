	RANQUEAMENTO EXTRAOFICIAL UFMS
Esse projeto é uma iniciativa pessoal para aprendizado e aprimoramento das habilidades de programação e lógica.

	COMO FUNCIONA
Após o término das inscrições, a UFMS disponibiliza uma lista com todos os inscritos no PSV daquele ano. Essa lista contém o número de inscrição do candidato, 
seu respectivo código e o curso escolhido. Após a realização da prova, a UFMS divulga as notas dos candidatos de maneira anonimizada, onde aparece apenas o número 
de inscrição e a nota correspondente. O código captura os números de inscrição e os códigos de curso da primeira lista da UFMS e cruza esses dados com a lista anonimizada. 
Em seguida, realiza-se o cálculo da média do candidato, levando em consideração os pesos definidos no edital do PSV. Por fim, é feita a classificação e o apontamento da nota de corte.
